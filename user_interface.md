## Shortcuts

**Default**

`lm` Left Mouse  
`mm` Middle Mouse  
`rm` Right Mouse  
`sm` Scroll Mouse  
`cm` Cursor Mouse  

`[` Up  
`]` Down  
`{` Left  
`}` Right  
`<` Page Up  
`>` Page Down
`\` Home  
`|` End  

`!` Alt  
`@` Backspace  
`#` Ctrl 
`$` Delete  
`%` Enter  
`&` Esc  
`*` Return  
`(` Shift   
`)` Tab   

**Properties**  
`#c` copy  
`#v` paste  
`#!c`  
`#!v`  
`rm` menu  
`@` clear  
`-`  
`*`  
`!`  

**Animation**  
`i` insert  
`!i` clear  
`(#i` clear all  
`#d` assign a driver  
`#!d` clear  
`k` add set  
`!k` clear set
